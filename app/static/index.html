ABOUT PLOTTERTOOL
SYSTEM OVERVIEW

Integrated toolpath optimization platform for CNC pen plotters. Converts vector graphics to machine-executable G-code with automated path reduction and overlap elimination.

★ VIEW SOURCE ON GITHUB

SETTINGS REFERENCE
Z UP / Z DOWN Pen lift and lower positions in mm. Detected automatically from G-code, adjust if needed.
PEN WIDTH Diameter of pen tip in mm. Used for overlap detection - lines hidden under other strokes are removed. Set to 0 to disable filtering.
MIN VISIBILITY Minimum visible percentage required to keep a line. At 50%, lines more than half-covered by other strokes are removed.
MERGE THRESHOLD Maximum gap in mm to merge adjacent paths. Eliminates pen lifts for tiny gaps (e.g., 0.5mm). Set to 0 to disable.
DRAW / TRAVEL / Z SPEED Feed rates in mm/min for drawing, rapid moves, and pen up/down movements respectively.
CURVE TOLERANCE SVG conversion accuracy in mm. Lower values = more points = smoother curves but larger files.
HEADER / FOOTER G-code commands added at start/end. Typically homing (G28) and parking sequences.
